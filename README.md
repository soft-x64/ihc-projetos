Este repositório reúne atividades práticas e projetos acadêmicos voltados para o estudo da interação entre usuários e sistemas digitais.

Aqui você encontrará:
- 📱 Protótipos de interfaces

- 🎨 Projetos de design de interação
- 💡 Soluções centradas no usuário
